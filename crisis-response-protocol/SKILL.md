# 危机应对协议 (Crisis Response Protocol)

本技能帮助你在AI辅导系统中实现安全的危机干预功能，遵循心理健康领域的最佳实践。

## 何时使用

### ✅ 使用此技能的场景：
- 在康复/心理健康应用中实施危机检测
- 构建安全计划功能
- 集成热线和紧急资源展示
- 设计风险评估界面
- 为AI聊天系统创建升级协议

### ❌ 切勿用于：
- **亲自应对真实危机** → 请拨打当地心理援助热线或紧急服务电话
- 一般心理健康内容 → 使用其他专注康复教练模式的技能
- 医疗建议或诊断 → 始终以专业人士意见为准
- 替代人类危机咨询师 → AI应起辅助作用，而非替代

## 核心安全原则

> **在急性危机情况下，AI绝不应成为唯一的响应者。**
> 必须始终提供通往人类支持和紧急服务的途径。

## 1. 危机检测

### 风险指标分类
通过分析用户输入中的关键词，对危机等级进行分类。

- **一级指标 (Critical)**：自杀倾向、自残意图、伤害他人意图、急性物质过量。
- **二级指标 (High)**：严重抑郁、惊恐发作症状、精神病性症状、严重焦虑。
- **三级指标 (Medium)**：孤立倾向、复发模式、绝望感。

## 2. 分级响应协议

- **级别 1 (None)**：正常AI互动。
- **级别 2 (Low)**：温和询问，提供grounding练习。
- **级别 3 (Medium)**：提供安全资源（12320-5, 400-161-9995）。
- **级别 4 (High)**：启动危机协议，强调寻求身边信任的人，提示当地急诊。
- **级别 5 (Critical)**：紧急升级，提示立即拨打 110 或去急诊室，系统标记人工审核。

## 3. 安全资源数据库

在使用此技能时，务必加载最新的安全资源数据，确保号码和链接准确：

- **全国心理援助热线**：12320-5
- **希望24热线**：400-161-9995
- **北京心理危机研究与干预中心**：010-82951332

## 4. AI安全护栏 (System Prompt Injection)

在处理聊天逻辑时，务必注入以下关键指令：
1. 你不是治疗师或危机咨询师。
2. 任何涉及自我伤害或暴力的内容：立即表达关切，提供热线资源，鼓励专业协助，**不进行深度咨询**。
3. 绝不轻视感受，不使用有毒正向思维。
4. 绝不对安全相关问题承诺保密。
